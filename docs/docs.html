Git VS GitHub

Git es un sistema que permite manejar información, aplicando control de versiones. "GitHub" es un sitio web que aloja código de sistemas, aplicando control de versiones al estilo Git. El control de versiones es clave al momento de construir, mejorar y mantener un sistema ya que gracias a éste podemos ir agregando funcionalidad de a poco y cuando se genere un error simplemente podemos volver en el tiempo a un estado seguro.

Comandos útiles
  • git clone
  • git pull
  • git status
  • git branch
  • git checkout
  • git add
  • git commit
  • git push

Cuando estamos trabajando en un mismo repositorio con otros colaboradores puede que nos surgan conflictos al hacer un "merge". Esos conflictos son detectados por Git y se muestran así:

El lenguaje HTML

HTML es un lenguaje; no de programación sino de estructura de una página web. Nos permite situar elementos dentro de la página siguiendo la lógica y estructura del modelo conocido como DOM(Document Object Model).
